Math.floor()

The Math.floor() function in JavaScript is used to return the largest integer less than or equal to a given number. Essentially, it "cuts off" the decimal part of a number and returns the integer part.

Here's an example of using Math.floor():

console.log(Math.floor(3.9));
console.log(Math.floor(5.7));
console.log(Math.floor(-2.3));

In each case, Math.floor() returns the largest integer less than or equal to the given number. So, for 3.9, the result is 3, for 5.7 it is 5, and for -2.3 it is -3.
